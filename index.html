Bootstrap Guide

Why Use Bootstrap?

  1. Bootstrap helps to build websites faster with ready-to-use classes.
  2. It comes with responsive grid system to design layouts easily.
  3. Provides pre-built components like navbar, cards, buttons, forms, modals.
  4. Supports cross-browser compatibility and mobile-first design.
  5. Syntax: <div class="row"><div class="col">Content</div></div>
Tip: Always use the grid system for layout. It keeps your website responsive!
Practice Tasks:
  • Create a Navbar with brand and menu items
  • Design a 3-column layout using grid
  • Add a responsive card with image + text
  • Style buttons using different Bootstrap classes
Remember:

✔ Use container or container-fluid for layouts
✔ Use row and col for grids
✔ Use bg-color classes for background
✔ Use text-color classes for text styles
✔ Bootstrap works best with mobile-first approach!
